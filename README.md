# JWT Decoder Plus (devtools)

A simple web-based tool for decoding JWT tokens and parsing JSON, built with React. This tool helps you:

- Decode and inspect JWT tokens visually
- Parse and pretty-print JSON with error highlighting
- View claim details, including expiration time with countdown

## Features
- Paste or type a JWT token to instantly decode and view its claims
- Paste or type JSON to validate and pretty-print it, with partial parsing for invalid JSON
- Expiration (`exp`) claim is highlighted for easy identification
- Responsive, modern UI

## Access the Tool
The app is published as a static site using GitHub Pages. You can access it here:

👉 **[Open JWT Decoder Plus](https://anoop0.github.io/devtools)**

## Usage
1. Open the link above in your browser.
2. Use the tabs to switch between JWT Decoder and JSON Parser.
3. Paste your JWT token or JSON in the input area.
4. View decoded claims or parsed JSON instantly.

## Local Development
To run or develop locally:

```bash
npm install
npm start
```

## Deploying to GitHub Pages
To build and deploy the latest version:

```bash
npm run deploy
```
